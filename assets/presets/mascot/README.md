# サンプル画像の置き場所（マスコット作成）

このフォルダに `<プリセットID>.png` を置くだけで、プリセットカードにサムネイル・
🔍拡大が自動で出ます（LP作成の `assets/presets/lp/` と同じ仕組み）。**コード変更は不要**。無くても崩れません。

見本カードには正面1枚で十分です（実際に使うときは同じ組み合わせで横・背面も生成すれば三面図が揃います）。
以下は各プリセットの実際の `imageSlots()` ロジックから生成した「正面」プロンプトです。
コピーして画像生成AI（ChatGPT Image 2.0 / Midjourney / DALL·E など）に貼ってください。
