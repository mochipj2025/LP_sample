# サンプル画像の置き場所（画像作成）

このフォルダに `<プリセットID>.webp` を置くだけで、プリセットカードにサムネイル・
🔍拡大が自動で出ます（LP作成の `assets/presets/lp/` と同じ仕組み）。**コード変更は不要**。無くても崩れません。

現状は絵文字アイコン止まりで、LP作成だけ本物の見本画像がある状態でした。ブランディングを揃えるため、
各プリセットの実際の `build()` ロジックから生成したプロンプトを以下に用意しています。
コピーして画像生成AI（ChatGPT Image 2.0 / Midjourney / DALL·E など）に貼り、できた画像をファイル名で保存してください。

| プリセット | ファイル名 | 比率 |
|-----------|-----------|------|
| SNS投稿用の1枚 | sns.webp | 4:5（縦長SNS） |
| 広告バナー | banner.webp | 16:9（横長・バナー） |
| サイトのヒーロー画像 | hero.webp | 16:9（横長・バナー） |
| 商品写真 | product.webp | 1:1（正方形） |
| 名刺・ショップカード | card.webp | 1:1（正方形） |
| OGP画像 | ogp.webp | 16:9（横長・バナー） |

拡張子は **webp → png → jpg → jpeg** の順で最初に見つかったものを使用
（`js/app.js` の `IMAGE_EXTS` と同じ順）。長辺720px・150KB以下推奨。

これらは「各プリセットを選んだときに実際に生成される文面」に、代表的な被写体（明るいカフェの1枚・化粧品ボトルなど）を
仮当てして作った実出力です。手書きの別文章ではなく、ツール本体の `buildOnePrompt()` から得たものなので、
実際にこのプリセットを使ったときの雰囲気とズレません。
