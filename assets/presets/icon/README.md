# サンプル画像の置き場所（アイコン作成）

このフォルダに `<プリセットID>.png` を置くだけで、プリセットカードにサムネイル・
🔍拡大が自動で出ます（LP作成の `assets/presets/lp/` と同じ仕組み）。**コード変更は不要**。無くても崩れません。

以下は各プリセットの実際の `build()` ロジックから生成したプロンプトです。
コピーして画像生成AI（ChatGPT Image 2.0 / Midjourney / DALL·E など）に貼り、できた画像をファイル名で保存してください。
出力は背景透過のPNGを想定していますが、カード上のサムネイルとしてはそのまま使えます。

| プリセット | ファイル名 |
|-----------|-----------|
| ファビコン・アプリアイコン | favicon.png |
| SNSプロフィールアイコン | sns.png |
| UIボタン・機能アイコン | ui.png |
| ロゴマーク | logo.png |
| グッズ・スタンプ用 | goods.png |

拡張子は **png → webp → jpg → jpeg** の順で最初に見つかったものを使用
（`js/app.js` の `IMAGE_EXTS` と同じ順）。いずれも正方形・長辺720px程度推奨。
